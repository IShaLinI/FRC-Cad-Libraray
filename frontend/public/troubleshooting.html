Troubleshooting

Most issues occur because of browser privacy settings. This is because the app exists in an iFrame inside Onshape and tries to access its own cookies, which is a common strategy that trackers use. Disable any privacy extensions such as Adblock, uBlock, Privacy Badger, etc, specifically for mkcad.julias.ch and cad.onshape.com if desired.
If you are on an enterprise network, you may need your IT services to allow mkcad.julias.ch through the firewall.

Specific browser info:
  • Firefox
  • Chrome
  • Safari
  • Nothing Worked

Firefox

  • Make sure privacy extensions are disabled, or try using the app in Incognito Mode.
  • Disable tracking protections for mkcad.julias.ch and cad.onshape.com.
  • Nuclear option: Disable all tracking protections.

Chrome

  • Make sure privacy extensions are disabled, or try using the app in Incognito Mode.
  • Nuclear option: Disable all tracking protection. Settings -> Privacy and Security -> choose No Protection

Safari

  • Make sure privacy extensions are disabled, or try using the app in Incognito Mode.
  • Disable "Prevent cross-site tracking"

Nothing Worked

File an issue on Github or Chief Delphi. Make sure to include your OS, browser (and version), and a description of the issue including what steps you've already tried.
